## 版本 C:开发副业向

### 标题

> 把别人 17 万 ⭐ 的项目重新打包,3 天写完,3 个上游邀请合作 🚀

### 正文

> 第一次做开源副业,流程分享给想做独立开发的家人们 ⬇️
>
> 起因:看上 Nous Research 的 Hermes Agent(17 万 ⭐),功能爆炸但安装步骤劝退非程序员。
>
> 思路:不是从零写一个,而是「在巨人肩膀上做一个 wrapper」——
>
> 把它的 Python 后端 + Vue 前端打包成桌面应用,目标用户:**想用 AI Agent 但不会命令行的人**。
>
> 关键决策:
>
> 📦 用 Electron(不用 Tauri)—— 因为 Vue 那部分本来就是 Node,只用 Node 一种运行时,而不是 Tauri 还要再嵌一份
>
> 🐍 用 python-build-standalone(Astral 出的)内嵌 CPython —— 用户机器不用装 Python
>
> 🛠️ 用 uv 装 hermes-agent(不用 pip)—— pip 在公司网络下经常挂死
>
> ⚙️ GitHub Actions matrix CI 自动构建 5 平台
>
> 🔄 electron-updater 接 GitHub Releases,版本一推用户自动更新
>
> 还顺手发现并修了 2 个上游 bug,顺便提了 PR、issue,可能后续会被 merge 进主项目 🎉
>
> 整个项目代码全开源 MIT,现在有人愿意用、有人开始下载,我觉得很满足。
>
> 独立开发不用一定要 from scratch,**「让别人的好东西更多人能用上」也是有价值的**。
>
> 想看代码的扣「2」,想要安装包的扣「1」,我私信。
>

## 发帖时间窗口

小红书算法对发帖时间敏感:

- **工作日**:中午 12:00-13:00、晚上 19:00-22:00 是金时段
- **周末**:全天都行,周日晚最佳
- 千万**别**在工作时间发(用户在上班没空看)

第一条**两小时内**互动率(点赞 / 收藏 / 评论 / 完播)决定它能不能进流量池。所以发完前两小时至少要把帖子转给 3-5 个朋友点个赞 + 评论。
